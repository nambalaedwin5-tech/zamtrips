# ✈️ ZamTrips Booking System

A Flask-based travel booking management system designed for tour operators.

## Features
- Booking management dashboard
- Create, edit, delete bookings
- Status tracking (Pending, Confirmed, Cancelled)
- Search functionality
- CSV export
- Login authentication
- Responsive Bootstrap UI

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS/Bootstrap
